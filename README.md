# Sandbag
Sandbag is an application for all climbing-lovers. You can check new, cool places all around the world to hike. Also you can check ranking of best climbers and most popular climbing places.

1. Clone our app:
```bash

```

2. Enter directory:
```bash
$ cd vol2-delta
```

```bash

```

5. Run rails server:
```bash
$ rails server
```
